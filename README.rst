=========
RSTEditor
=========
RSTEditor is a editor for ReStructedText. It support preview with html.

Requirement
===========
Install them on all platform:

+ docutils
+ wxPython
+ Pygments

For Window:

.. tip::

    Do not support scroll synchronize on Window.

Install it on Window:

+ comtypes

For Linux:
Install them:

+ pygtk
+ pywebkitgtk

========
